This module allow to import product and their varients with current stock on a single click

Please go to link to see functionality of this module: Click To View

Functionality:-

-> This module allows to import product variants and their current stock.
-> We can either import product variant stock individual or same stock for all variants.
-> As shown in image there is a csv file in module with same header or we can use our own csv but header should be same
-> This image represent the same stock for all variants.
-> After compleation of import script this will import 2 qty on hand per variants(Yellow,Blue and Red).

-> If we wants to import Stock per variants then we have to define per variants in individual lines as shown in image


-> Now go to Inventory -> Import Products and click on Import Products


-> It will openup the popup (Wizard) in that upload the csv file and click on import as shown in image.


-> If all data import successfull then it will give a message as per shown in image.


-> Now can check the product as it updated as per data fill on script.
