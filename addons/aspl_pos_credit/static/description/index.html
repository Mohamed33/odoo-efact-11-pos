Odoo 11 POS Customer Credit And Debt Payment

Allows user pay partially for POS orders.


Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • This module is mainly used to do the partial payment for POS orders.
    For example: Lets say a customer comes to the shop and makes an order total of $200 but he wants to pay $100 as half payment and remaining due, the very next day. With the help of this module, he can do so.
  • User can also view past POS orders, Reorder the particular drafted order and make an order as draft from POS. It is also possible to re-print receipt and view ordered products from POS Order List. There are some filtrations based on state, date of order and Customer.
  • User can give credit to customer and customer can pay by credit.

How to use

  • As per below image user can set configuration to enable this feature.




  • User can pay partial payment of POS order.




  • In back-end entry of pos order in new state.


  • User can pay the due amount of POS order and also give credit amount.




  • If user pay full amount of POS order then entry of POS order in paid state.




  • User can pay POS order amount using the credit amount.
