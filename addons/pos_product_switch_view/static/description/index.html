POS Product Switch View

This module helps to switch the products view (Form/List).

Point of Sale Configuration

Click on Point of Sale menu, go to the point of sale in sub-menu of configuration. This layout will show and user can switch the view by "Product Switch View". Bydefault selected view will show in the Point of sale screen.

List View

Products will show same like this view.

Form View

Products will show same like this view.
